Учебный лендинг с интерактивными демо по аномалиям обработки HTTP-запросов, включая обход маршрутизации первого запроса (first-request routing bypass) и другие нетривиальные кейсы.

Открыть лабораторию Демки для разбора аномалий маршрутизации, заголовков и балансировщиков.

Используйте эту лабораторию как песочницу для исследования поведения прокси, балансировщиков и бэкендов при нестандартных HTTP‑последовательностях.
